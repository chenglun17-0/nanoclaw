# 数据存储模块（SQLite）

[返回模块索引](../MODULES.md)

## 模块定位

提供统一持久化能力，覆盖消息、聊天、任务、路由状态、群注册和会话映射。

## 关键文件

- `src/db.ts`

## 主要职责

1. 初始化 schema 并兼容历史迁移。
2. 提供消息、任务、群组、session 的读写接口。
3. 保存路由游标和运行状态。
4. 为调试/管理提供查询接口（如聊天列表、任务列表）。

## 输入与输出

- 输入：上层业务调用参数。
- 输出：持久化结果、查询记录、迁移副作用。

1. schema 变更必须保留迁移兼容逻辑。
2. 若新增状态字段，需评估是否需要在启动时回填或容错。
