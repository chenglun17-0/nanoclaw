# 安全与挂载校验模块

[返回模块索引](../MODULES.md)

## 模块定位

定义并执行额外挂载安全策略，确保容器只能访问允许路径且遵守读写限制。

## 关键文件

- `src/mount-security.ts`

## 主要职责

1. 从外部 allowlist 加载允许挂载根目录。
2. 默认阻断敏感路径模式（如 `.ssh`、`credentials`、`.env` 等）。
3. 校验路径规范性、真实路径、越界与软链接风险。
4. 根据 main/non-main 组自动调整只读策略。

## 输入与输出

- 输入：group 的 `additionalMounts` 请求。
- 输出：验证后的挂载清单或拒绝错误。

## 上下游关系

- 上游： [容器执行层](./05-container-execution.md)
- 下游：无（安全校验底层）

## 改动建议

1. 放宽白名单规则前要先做威胁评估和回滚方案。
2. 新增 blocked pattern 时，注意误伤正常业务目录。
